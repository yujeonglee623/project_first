# 받아야 하는 데이터
## 내부데이터 (ODM 기업으로부터 직접 확보)
### 데이터 유형 -> 데이터 수집/확보 방법 -> 유의사항 및 팁
- ODM 생산/매출/이익 데이터
    - ERP/MIS 시스템 연동 또는 DB 직접 접근 (API)
        - 필수: 고객사별, 제품별 원가, 판매가, 이익률 필드 정의
        - 가장 중요: 데이터 추출 전, 고객사와의 기밀 유지 협약(NDA) 체결 필수
- 제품 포뮬러/R&D 데이터
    - R&D 관리 시스템(PLM) 또는 규격서 DB 연동
        - 성분명, INCI명(국제 화장품 성분명), 제형, 기능 필드를 정형화해야 머신러닝 모델에 사용 가능
        - 성분/규격 데이터의 최신성을 유지하는 것이 핵심
- 수출 및 물류 데이터
    - 물류 시스템(WMS) 또는 무역 대행사 데이터
        - 국가별 통관일, 운송비용, 리드타임을 시계열로 수집
        - 수출 국가 코드(ISO 코드)를 통일하여 외부 데이터와 연동 준비

## 외부데이터
1. 공공/정부기관 데이터 (무료 or 저가)
    - 한국 관세청, 무역통계진흥원: 우리나라 화장품의 국가별, 품목별 수출입 통계 확인
    - 식품의약품안전처(식약처): 연간 화장품 생산 및 수출 실적 통계를 제공하여 시장 규모와 유형별 동향 파악하는데 도움
    - KOTRA (대한무역투자진흥공사): 주요 국가별 시장 리포트나 소비재 수출금액 데이터 등을 확보 가능
2. 상용 데이터 및 API (유료)
    - 글로벌 이커머스/검색량 데이터(API)
        - Google Trends API: 특정 국가/기간의 화장품 관련 키워드 검색량 추이 분석.
        - 소셜 미디어 분석 툴: 인스타그램, 틱톡 등에서 특정 성분/제품의 언급량 및 긍정/부정 반응을 수집하는 API (예: 뷰티 전문 데이터 분석 솔루션)
    - 시장 조사 기관 리포트
        - Mintel, Euromonitor, Statista 등의 기관에서 발간하는 국가별 뷰티 시장 리포트를 구매하여 정성적/정량적 정보를 확보. (비싸지만 심층적인 정보 제공)
    - 특정 뷰티 플랫폼 데이터
        - 화해, 올리브영, 세포라 등 주요 플랫폼의 리뷰 및 인기 제품 랭킹 데이터를 스크래핑(합법적인 범위 내에서)하거나, 협의를 통해 데이터 파트너십 구축.
